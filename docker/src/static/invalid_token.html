Invalid Client

Zoho Analytics MCP Server authentication error

Authentication failed: The registered client appears to have expired or is no longer valid.

To resolve this issue, remove the cached client credentials from your MCP Host and re-authenticate.

VS Code

Open the Command Palette (Ctrl + Shift + P) and run: 

Authentication: Remove Dynamic Authentication Providers

Cursor

Open the Command Palette (Ctrl + Shift + P) and run: 

cursor: clear all mcp tokens

Claude Desktop (MCP-REMOTE)

Delete the following directory and restart Claude Desktop: 

~/.mcp-auth

Next Steps

  • Restart your MCP Host after clearing credentials
  • Re-authenticate when prompted
  • Reconnect to the Zoho Analytics MCP Server
